AI-Powered Website Layout Generator:

This project demonstrates the use of AI to rapidly prototype website layouts. By providing a simple natural language description (e.g., "a website for a local bakery with a focus on cakes"), the system generates a basic HTML structure, giving developers a quick starting point for their projects.

Key Features:

  • AI-Driven HTML Generation: Uses OpenAI's GPT models to interpret user descriptions and create corresponding HTML.
  • Iterative Refinement: Allows users to progressively refine the generated layout by requesting more details or specific features.
  • Backend API (Flask): A Python Flask backend handles communication with the AI and manages data exchange.
  • Frontend Interface (HTML, CSS, JavaScript): A clean and intuitive user interface built with standard web technologies.

Tech Stack: Python (Flask), OpenAI API (GPT-3.5-turbo), HTML, CSS, JavaScript

YouTube Comments Analyzer:

This Flask web app lets you analyze YouTube comments. Enter a video URL, and it grabs comments using the YouTube Data API. It then stores them in an SQLite database and generates a word cloud visualization using the WordCloud library. You can even search through the comments!

Tech Stack:

  • Backend: Python, Flask
  • API: YouTube Data API
  • Database: SQLite
  • Visualization: WordCloud (Python library)
  • Frontend: HTML, CSS, JavaScript

Drone Delivery API:

This Drone Delivery API offers a comprehensive solution for managing and tracking drone deliveries. It leverages a Python Flask server to handle backend logic, storing crucial information about drones and pilots, and integrates with Google Maps API for real-time drone tracking visualized on a vanilla HTML and JavaScript frontend.
