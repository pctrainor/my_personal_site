Project Showcase: PMP Flashcards

To help me prepare for the Project Management Professional (PMP) exam, I recently developed a comprehensive flashcard application using JavaScript and JSON. This project was instrumental in my successful completion of the PMP exam this month.

Key Features:

  • Extensive Question Bank: The application boasts a collection of over 250 PMP exam sample questions covering all knowledge areas of the PMBOK Guide.
  • Formula Breakdown: A dedicated section provides a clear and concise breakdown of all essential mathematical formulas required for the PMP exam.
  • Interactive Learning: The flashcard format encourages active recall and spaced repetition, proven learning techniques for effective knowledge retention.
  • Generative AI Integration: Leveraging the power of generative AI, I was able to efficiently create a diverse and challenging set of practice questions.

This project demonstrates my ability to:

  • Apply technical skills: I utilized JavaScript and JSON to build a functional and user-friendly web application.
  • Solve real-world problems: I identified a personal need for effective study materials and developed a solution to address it.
  • Utilize AI for content creation: I harnessed the capabilities of generative AI to streamline the question creation process.
  • Achieve goals: The successful completion of my PMP exam highlights my dedication to personal and professional development.

This project is a testament to my passion for project management and my commitment to continuous learning. It showcases my ability to combine technical skills with creative problem-solving to achieve desired outcomes.
